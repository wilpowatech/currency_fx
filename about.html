About This Project

Kwame Currency Exchange is a smart web-based tool built to simplify and automate currency recognition and real-time exchange rate conversion.

This system combines computer vision and neural networks to identify physical currency notes and instantly display their value in the desired target currency.

It is designed for use in educational institutions, forex bureaus, mobile agents, and fintech developers who require lightweight, real-time currency solutions using modern APIs.

Real-time rates are powered by exchangerate.host. This project supports NGN, USD, EUR, AUD, INR, and GBP.

This application is part of the final year thesis of Kwame Tari Omonibo, and integrates frontend development with external APIs, UI design principles, and dynamic data visualization.
